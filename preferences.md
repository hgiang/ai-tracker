# Reader Preferences

> Edit this file to teach the digest what you find useful. The LLM filter reads
> this verbatim each time it re-ranks the daily candidate pool.

## About me

ML engineering manager focused on NLP and transformers in production: fine-tuning
and distilling large models down to millions of parameters for low-latency
serving (millisecond response, 10k+ QPS). Learning the broader AI-engineering
ecosystem (agents, RAG, evals, tooling) hands-on.

## Topics I want to see

- Large language models: new model releases, capability benchmarks, evals
- Agentic AI: tool use, multi-agent patterns, agent harnesses, evaluation
- Retrieval-Augmented Generation: embeddings, vector DBs, retrieval quality
- Fine-tuning and post-training: SFT, RLHF/DPO, LoRA, distillation
- Inference efficiency: quantization, speculative decoding, serving stacks
- Open-source models and tooling that I could realistically adopt at work

## Topics to skip

- Crypto / web3 / NFT crossovers
- Generic AI hype, listicles, "10 prompts to..." content
- Pure marketing announcements with no technical substance
- Image/video generation models, unless the technique transfers to LLMs
- Career advice, opinion pieces, founder think-pieces

## Examples of useful items

- A technical write-up of a new distillation technique with benchmarks
- A small open-source project that shows a clean agentic pattern
- A paper that proposes a measurable improvement on a serving bottleneck
- An honest evaluation comparing two model families on a real task

## Examples I'd skip

- "OpenAI announces..." press releases with no detail
- Job postings or hiring threads
- Generic "the future of AI" essays
- Tutorials on prompts that aren't backed by evals
